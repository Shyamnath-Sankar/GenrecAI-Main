# GENREC Website - Issues Report
Date: March 27, 2026

## CRITICAL ISSUE: Broken Admin Blog Links

### Problem
The Blog Manager admin panel has broken "Edit" and "New Post" links. When clicking these buttons, the user is redirected to the homepage instead of the blog editor.

### Root Cause
In file: `D:\Projects\websites\genrecwebsite\pages\admin\BlogManager.tsx`

The links are using the wrong path prefix:
- Current (INCORRECT): `/admin/blogs/...`
- Expected (CORRECT): `/adminofthegenrecai/dashboard/blogs/...`

### Broken Links Found (Lines 72, 88, 139):
1. Line 72: `to="/admin/blogs/new"` 
   - Should be: `to="/adminofthegenrecai/dashboard/blogs/new"`
   
2. Line 88: `to="/admin/blogs/new"`
   - Should be: `to="/adminofthegenrecai/dashboard/blogs/new"`
   
3. Line 139: `to={\`/admin/blogs/edit/${blog.id}\`}`
   - Should be: `to={\`/adminofthegenrecai/dashboard/blogs/edit/${blog.id}\`}`
